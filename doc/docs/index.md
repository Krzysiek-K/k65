# Welcome to K65

K65 is a compiler for [6502 CPU](https://en.wikipedia.org/wiki/MOS_Technology_6502) architecture. At this moment it supports **Atari 8-bit** (400/800/XL/XE), **Commodore 64** and **Atari 2600** as target platforms.
